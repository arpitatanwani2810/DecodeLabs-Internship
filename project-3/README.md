# AI Recommendation System

## Overview

This project implements a simple content-based recommendation system using Python. The system accepts user interests, compares them with predefined item attributes, calculates similarity scores, and recommends the most relevant items.

## Features

* User preference input
* Similarity-based recommendation logic
* Automatic ranking of recommendations
* Top 5 personalized recommendations
* Recommendation score visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## How It Works

1. The user enters their interests.
2. The system compares those interests with item tags.
3. A similarity score is calculated for each item.
4. Items are ranked based on relevance.
5. The top recommendations are displayed and visualized.

## Sample Input

```text
ai, python, data
```

## Output

The system displays the most relevant recommendations along with their similarity scores and generates a visualization showing recommendation rankings.

## Learning Outcomes

* Recommendation Systems
* Content-Based Filtering
* Pattern Matching
* Similarity Analysis
* Data Visualization

## Conclusion

This project demonstrates the core concepts behind recommendation systems by matching user preferences with item attributes and generating personalized suggestions using algorithmic logic.
